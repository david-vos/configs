# What is this repo?
This repo contains most of my random config files that I used to setup whatever. The names of file will be how they are suppose to named in the application.
> Of course I will put files that should be named `config.whatever` inside a folder that that is named like the application.

# How to use 
This repo could be used to get inspiration about your own configs. I do not want to support or create installers/universal config files.
> If you see something that you like that you would like to copy but don't know how? Just send me a message and I'll try to help you the best that I can!
